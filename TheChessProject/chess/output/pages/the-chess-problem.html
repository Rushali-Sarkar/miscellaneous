The Chess Problem

Welcome to The Chess Problem!

This a project in which any PGN file is read and respective board view after every move is shown.

To know about the rules of chess refer to [RULES][./Rules/rules.md]

To know about the PGN representation refer to [PGN][./Pgn/pgn.md]

To view the step by step solution of the problem refer to [SOLUTIONS][./Solutions/solution.md]

I have tried to solve the entire problem using python. I have tried to make the solution as readable as possible. Hope you Enjoy It!!
